1.3. Changelog

This is version 1.0-musl of the Linux From Scratch book, dated November 1, 2025. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via https://www.linuxfromscratch.org/mirrors.html.
