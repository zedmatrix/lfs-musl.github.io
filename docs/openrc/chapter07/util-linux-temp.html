4.10. Util-linux-temp-2.41.3

The Util-linux package contains miscellaneous utility programs.

Package Information

  • Download HTTP: https://www.kernel.org/pub/linux/utils/util-linux/v2.41/util-linux-2.41.3.tar.xz

  • Download SHA256 Sum: 3330d873f0fceb5560b89a7dc14e4f3288bbd880e96903ed9b50ec2b5799e58b

  • Download Size: 9,245 KB

  • Estimated Disk Space: 128.9M

  • Estimated Build Time: less than 0.5 SBU

4.10.1. Installation of Util-linux-temp

Prepare Util-linux-temp for compilation:

mkdir -pv /var/lib/hwclock

./configure --libdir=/usr/lib \
            --runstatedir=/run \
            --disable-chfn-chsh \
            --disable-login \
            --disable-nologin \
            --disable-su \
            --disable-setpriv \
            --disable-runuser \
            --disable-pylibmount \
            --disable-static \
            --disable-liblastlog2 \
            --without-python \
            ADJTIME_PATH=/var/lib/hwclock/adjtime \
            --docdir=/usr/share/doc/util-linux-2.41.3

To Compile the Package, issue:

make CPPFLAGS="-DTPARM_VARARGS" LIBS="-lterminfo -lcurses"

Now Install the Package, as the root user:

make install
