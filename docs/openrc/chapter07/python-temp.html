4.7. Python-temp-3.14.3

The Python 3 package contains the Python development environment.

Package Information

  • Download HTTP: https://www.python.org/ftp/python/3.14.3/Python-3.14.3.tar.xz

  • Download SHA256 Sum: a97d5549e9ad81fe17159ed02c68774ad5d266c72f8d9a0b5a9c371fe85d902b

  • Download Size: 23,221 KB

  • Estimated Disk Space: 327.1M

  • Estimated Build Time: 1.70 SBU

4.7.1. Installation of Python-temp

Prepare Python-temp for compilation:

./configure --prefix=/usr \
        --enable-shared \
        --without-ensurepip \
        --without-static-libpython \
        --enable-optimizations \
        --disable-test-modules

To Compile the Package, issue:

make

Now Install the Package, as the root user:

make install
