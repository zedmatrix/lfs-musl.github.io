3.2. Netbsd-curses-temp-0.3.2

this is a port of netbsd's curses library for usage on Linux systems (tested and developed on sabotage linux, based on musl libc).

Package Information

  • Download HTTP: http://ftp.barfooze.de/pub/sabotage/tarballs/netbsd-curses-0.3.2.tar.xz

  • Download SHA256 Sum: d28358b833e6cabcf229fecef98c649e2e90c29858aa34220476faf0e5ce10a1

  • Download Size: 522 KB

  • Estimated Disk Space: 4.6M

  • Estimated Build Time: less than 0.5 SBU

3.2.1. Installation of Netbsd-curses-temp

Prepare Netbsd-curses-temp for compilation:

To Compile the Package, issue:

make HOSTCC="cc" CC="$LFS_TGT-gcc -DTERMINFO_COMPAT" PREFIX="/usr" DESTDIR=$LFS all-dynamic

Now Install the Package, as the root user:

make HOSTCC="cc" CC="$LFS_TGT-gcc -DTERMINFO_COMPAT" PREFIX="/usr" DESTDIR=$LFS install-dynamic
