3.3. Bash-temp-5.3

The Bash package contains the Bourne-Again Shell.

Package Information

  • Download HTTP: https://mirror.csclub.uwaterloo.ca/gnu/bash/bash-5.3.tar.gz

  • Download SHA256 Sum: 0d5cd86965f869a26cf64f4b71be7b96f90a3ba8b3d74e27e8e9d9d5550f31ba

  • Download Size: 11,090 KB

  • Estimated Disk Space: 52.9M

  • Estimated Build Time: less than 0.5 SBU

3.3.1. Installation of Bash-temp

Prepare Bash-temp for compilation: 

./configure --prefix=/usr \
        --build=$(sh support/config.guess) \
        --host=$LFS_TGT \
        --without-bash-malloc

To Compile the Package, issue: 

make

Now Install the Package, as the lfs user: 

make DESTDIR=$LFS install &&
ln -sv bash $LFS/bin/sh
