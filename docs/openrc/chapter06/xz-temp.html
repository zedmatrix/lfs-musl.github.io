3.14. Xz-temp-5.8.3

The Xz package contains programs for compressing and decompressing files.

Package Information

  • Download HTTP: https://github.com//tukaani-project/xz/releases/download/v5.8.3/xz-5.8.3.tar.xz

  • Download SHA256 Sum: fff1ffcf2b0da84d308a14de513a1aa23d4e9aa3464d17e64b9714bfdd0bbfb6

  • Download Size: 1,512 KB

  • Estimated Disk Space: 16.5M

  • Estimated Build Time: less than 0.5 SBU

3.14.1. Installation of Xz-temp

Prepare Xz-temp for compilation: 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(build-aux/config.guess) \
            --disable-static \
            --docdir=/usr/share/doc/xz-5.8.3

To Compile the Package, issue: 

make

Now Install the Package, as the lfs user: 

make DESTDIR=$LFS install
rm -v $LFS/usr/lib/liblzma.la

3.14.2. Contents of Xz-temp

Installed programs: xz, xzdec, xzdiff, xzgrep, xzmore, and xzless
Installed libraries: liblzma.so.5.8.3
Installed directories: $LFS/usr/share/doc/xz-5.8.3
