2.4. Musl-temp-1.2.6

The Musl Package contains an alternative main C library.

Package Information

  • Download HTTP: https://musl.libc.org/releases/musl-1.2.6.tar.gz

  • Download SHA256 Sum: d585fd3b613c66151fc3249e8ed44f77020cb5e6c1e635a616d3f9f82460512a

  • Download Size: 1,057 KB

  • Estimated Disk Space: 25.2M

  • Estimated Build Time: less than 0.5 SBU

2.4.1. Installation of Musl-temp

Prepare Musl-temp for compilation:

./configure --prefix=/usr \
        --target=$LFS_TGT \
        --build=x86_64-lfs-linux-musl \
        --with-malloc=rpmalloc \
        --enable-debug

To Compile the Package, issue:

make

Now Install the Package, as the root user:

make DESTDIR=$LFS install
echo 'int main(){}' | $LFS_TGT-gcc -xc - ;
readelf -l a.out | grep ld-musl
