2.4.1. Installation of Musl-temp

Prepare Musl-temp for compilation: 

./configure --prefix=/usr \
        --target=$LFS_TGT \
        --build=x86_64-lfs-linux-musl \
        --with-malloc=rpmalloc \
        --enable-debug

To Compile the Package, issue: 

make

Now Install the Package, as the lfs user: 

make DESTDIR=$LFS install
echo 'int main(){}' | $LFS_TGT-gcc -xc - ;
readelf -l a.out | grep ld-musl
